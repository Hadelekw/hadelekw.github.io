MPlusA

A small Python library for calculations in tropical algebra.

Installation

MPlusA was initially created in Python 3.11 and was developed to be independent from any operating system specifics. It is believed that the library would work for previous versions of Python3 but it was not tested.

To install MPlusA in your Python virtual environment use pip:

Bash

$ pip install mplusa

Function list

add(*args)

This function is essentially an alias for min Python function. It also performs the necessary checks of the domain.

Arguments:

  • *args: Any numerical arguments in the tropical semiring domain.

Returns:

  • The minimum.

mult(*args)

This function is essentially an alias for sum Python function. It also performs the necessary checks of the domain.

Arguments:

  • *args: Any numerical arguments in the tropical semiring domain.

Returns:

  • The sum.

power(a, k)

This applies the mult function k times. Currently it only allows integer powers to be used. It is equivalent to the following expression: $$ \left(a_{\otimes}^{k} = \bigotimes_{i=1}^{k}a\right) \equiv \left( k \cdot a = \sum_{i=1}^{k}a\right)$$

Arguments:

  • a: Any numerical argument in the tropical semiring domain.
  • k: An integer to be used as an exponent.

Returns:

  • The product.

modulo(a, t)

This function is based on the definition in [1]: $$\forall a,t \in [0, \infty) \exists b \in [0, \infty): a \diamond t = b \iff a = b \otimes t_{\otimes}^{k}$$ $$\forall a \in [0, \infty): a \diamond t = a; \quad \forall a \in [e, \infty): \infty \diamond a = \infty$$

Arguments:

  • a: Any positive numerical argument in the tropical semiring domain.
  • t: An integer to be used as a divisor.

Returns:

  • The difference between the largest.

add_matrices(A, B)

Perform the parallel composition (sum) of equally-sized matrices A and B. This operation is defined as: $$\left(\mathbf{A} \oplus \mathbf{B}\right)_{ij} = \mathbf{A}_{ij} \oplus \mathbf{B}_{ij}$$ which is equivalent to the definition of this operation in linear algebra.

Arguments:

  • A, B: Numpy array objects with the items within the tropical domain and of equal sizes.

Returns:

  • A new numpy array object of size equal to the size of the arguments.

mult_matrices(A, B)

Perform the series composition (product) of matrices A and B with sizes m by n and n by p respectively. This operation is defined as: $$\left(\mathbf{A} \otimes \mathbf{B}\right)_{ij} = \bigoplus_{k=1}^{n} \left(\mathbf{A}_{ik} \otimes \mathbf{B}_{kj}\right)$$ which is equivalent to the definition of this operation in linear algebra.

Arguments:

  • A, B: Numpy array objects with the items within the tropical domain and of the right sizes.

Returns:

  • A new numpy array object of size equal to m by p.

power_matrix(A, k)

This applies the mult_matrices function k times. Currently it only allows integer powers to be used.

Arguments:

  • A: A numpy array object with the items within the tropical domain.
  • k: An integer to be used as an exponent.

Returns:

  • A new numpy array object of size equal to the input.

modulo_matrices(A, b)

This calculates the modulo of an n by m matrix A and an n-element vector b. This is by definition from [1]: $$\mathbf{B}_{ij} = \mathbf{A}_{ij} \diamond \mathbf{b}_i$$ where B is an n by m matrix.

Arguments:

  • A: A numpy array object with the items within the tropical domain.
  • b: A numpy array object with the items within the tropical domain. It has to be 2-dimensional and of size corresponding to the definition of A.

Returns:

  • A new numpy array object of size equal to the size of A.

unit_matrix(width, height)

Returns a 2-dimensional tropical unit matrix of given size.

Arguments:

  • width: the width of the matrix.
  • height: the height of the matrix.

returns:

  • A new numpy array object of given size.

star(A)

An unitary operator defined on a square matrix: $$\mathbf{A}^{*} = \bigoplus_{k \in \mathbb{N}} \mathbf{A}_{\otimes}^{k}$$

Arguments:

  • A: A square numpy array object with the items within the tropical domain.

Returns:

  • A new numpy array object of size equal to the input.

Class list

Polynomial(*coefficients)

A class which representing a tropical polynomial with one variable.

Functions:

  • __call__(x): return the value of the polynomial for a given point.
  • get_lines(): returns the a and b values of standard linear functions building the polynomial in form of y = ax + b.
  • get_hypersurface(): returns a list of the polynomial's roots.

Example code

Python

import mplusa.minplus as mp
import numpy as np

A, B = np.random.rand(5, 5), np.random.rand(5, 5)
C = mp.add_matrices(A, B)
